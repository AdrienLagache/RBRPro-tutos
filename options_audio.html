Volume du copilote (Navigator add-on)

Rien de bien compliqué ici, il s'agit du volume du copilote mais seulement si vous utilisez un copilote de type "Navigator". Vous pourrez également régler le volume directement dans l'addon Navigator.

Traitement audio/canaux audio

IndirectSound permet d'améliorer la position 3D de l'audio sur les jeu anciens, si vous possédez une version récente de Windows.
Attention, cela peut coûter des performances.

Le nombre de canaux audio à utiliser peut être définit sur automatique en sélectionnant 0. C'est ce que je vous conseille.

Reconnaissance vocale

Choisissez si vous voulez utiliser un bouton pour parler ou être en écoute permanente. Ceci fonctionne avec la reconnaissance vocale que vous pouvez activer en cochant le symbole "micro" en bas du manager.

Vous pourrez trouver une liste des commandes vocales qui fonctionne dans le manager et dans les menus du jeu dans le manuel de l'utilisateur. En ce qui concerne les commmandes vocales pour Navigator, vous trouverez un tuto dans la rubrique Navigator de ce site.

Seuil de glissement du pneu

Le son dû aux pneus dans Richard Burns Rally peut parfois être désagréable ou trop sensible. Ici vous pouvez régler le seuil miminum au dessus duquel vous entendrez glisser le pneu. Il ne s'agit que du son, évidemment.

Bande son

Grace à ce menu vous allez pouvoir changer la musique que vous entendrez dans les menus du jeu. Il suffit de les mettre dans le dossier qui s'ouvrira en cliquant sur "Ouvrir le fichier". Attention, votre musique devra être au format OGG Vorbis.
