Véhicules

Ici, vous pourrez créer vos listes de véhicules.
Attention, vous ne devrez mettre les tractions avant (FWD) que dans l'emplacement numéro 2.

Il existe quelques autres options comme installer les livrées ou les configurations pro, qui sont disponibles sur certains véhicules.
Vous pouvez également choisir d'installer toutes les voitures mais cela n'est pas forcément nécessaire, vous pouvez aussi les installer une par une.

Etapes

Même principe que pour les véhicules, vous avez ici de quoi créer 4 listes pour vos spéciales.
Une fois en jeu vous ne pourrez accéder qu'aux spéciales de votre liste. Heureusement vous pouvez en créer une avec toutes les spéciales en cliquant sur "sélectionner toutes les étapes".

Il est possible de télécharger toutes les spéciales d'un coup mais sachez que cela prendra beaucoup de place sur votre dique dur.
Si vous souhaitez tout de même les télécharger, vous pouvez cliquer sur le lien en haut ou tout simplement les ajouter à votre liste et démarrer le jeu. En effet les spéciales qui n'aurait pas été téléchargées et qui sont présentes dans votre liste se téléchargerons automatiquement au lancement du jeu.

Enfin, chaque spéciale possède un numéro. C'est ici que vous pourrez le trouver.

Pack d'options

Ici vous trouverez les différents plugins, patches, physics, FixUp, packs de texture,...

Il faudra vérifier que tout est bien à jour. C'est également ici que vous pourrez récupérer vos fichiers en cas de suppression par votre antivirus.

Certaines options s'installent automatiquement et d'autres ne sont pas compatibles avec tous les modes. Je vous conseille de bien lire les notes qui apparaissent à droite lorsque vous sélectionnez une option. Tout est expliqué et plutôt clair.

Add-ons

C'est ici que vous pouvez installer ou mettre à jour vos add-ons.

  • Navigator : il s'agit d'un addon permettant d'avoir des copilotes dont les notes seront plus détaillées et plus facilement modifiable. Vous pourrez même créer votre propre copilote avec votre voix ou une voix pré-enregistrée avec Navigator-builder. Nous détaillerons tout ça dans un chapitre dédié sur ce site mais en attendant, je vous recommande de lire le manuel de l'utilisateur Navigator. Vous pouvez également suivre la vidéo tutorielle faite par Carlo (les sous titres en français sont disponibles).
  • Pro light Plugin : Nous avons vu comment activer cette option dans la partie "Options/Video". Après l'installation, vous devriez pouvoir voir le menu RBRPro Lights apparaître dans les options du jeu en allant dans "plugin". Si il n'apparaît pas vous devrez réinstaller directX 9.0c. Vous pouvez utiliser ce lien pour le télécharger.
  • WRC Pro Dashboard : un nouveau tableau de bord plus complet que ceux de base. Il permet d'afficher, en plus des fonctionnalités classiques, les delta (temps, accélération, frein) par rapport à votre meilleur temps, vos actions sur les pédales, votre nom ainsi que celui de votre équipe, le logo PRO et celui du championnat, et même une présentation de l'équipage en début de spéciale. Il existe un manuel de l'utilisateur WRC Pro Dashboard, je vous invite à le consulter. Vous trouverez notamment la version .NET FRAMEWORK à télécharger si vous ne possédez pas au moins windows 10.
  • Telemetry Recorder/Router : Telemetry recorder vous permet de faire marcher les fonctionnalité de RBRPro qui nécessite des mesures, comme par exemple le WRC dashboard ou les hotlaps.
    Telemetry router, quant à lui, servira pour les logiciels ou add-on que vous souhaiteriez utiliser en plus de ceux présents dans RBRPro. Par exemple fanaleds.
  • Event Maker : cet outil n'est utile que pour les créateurs de championnat. Vous ne pourrez pas l'utiliser si vous n'êtes pas collaborateur.
  • Car Setup Installer : c'est un gestionnaire de configuration pour les réglages de vos voitures. En effet, il est parfois fastidieux de changer les réglages en jeu car vous ne pouvez pas entrer directement une valeur. Grâce à cet outil vous pourrez le faire à partir du manager en cliquant sur "gestionaire de configuration" dans Contenu/Véhicule
    Cliquez pour lire le manuel de l'utilisateur Car Setup Installer

Copilotes

Plusieurs choses à savoir ici :

  • Vous devrez faire attention à avoir la même langue dans le manager et dans les options "Notes" dans le jeu.
  • Il existe deux types de copilotes. Les copilotes standard, que vous pourrez modifier grâce au Pacenote Plugin (double-clic gauche en jeu) et aux options "Notes" du jeu. Les copilotes Navigator que vous pourrez configurer grâce à l'add-on Navigator. Il faudra être Pro-donateur (15 euros) pour pouvoir l'utiliser.
  • Il existe plusieurs systèmes de notation (Angle, difficulté, heures,...). A vous de voir avec lequel vous êtes le plus à l'aise.
