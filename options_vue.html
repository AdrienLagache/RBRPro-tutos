Vue en voiture

Ici, vous pouvez régler le FOV ainsi que la position du siège. En mode automatique ou semi-automatique, la caméra se place automatiquement mais cette option ne fonctionne pas sur toutes les voitures.

Attention, si vous voulez régler manuellement la vue vous devrez sélectionner "personnalisé" dans le menu déroulant puis en jeu, faire un double-clic droit afin de régler la caméra. Que vous souhaitiez changer de caméra ou simplement la déplacer, il faudra impérativement, soit terminer la session, soit quitter en abandonnant. Si vous quittez en sélectionnant "Menu principal" le réglage ne sera pas sauvegardé.

En VR, vous pouvez utiliser les touches 2,4,8,6 de votre clavier numérique pour modifier la vue et 0 pour la recentrer. Les modifications sont enregistrées automatiquement.

Caméras personnalisées

Vous pouvez configurer deux caméras pour les utiliser dans le jeu.

Dans le cas ou vous voudriez utiliser la caméra capot (cam_bonnet), vous devrez sélectionner "Custom 1" ET "Custom 2" dans "Camera 1" et "Camera 2".

Volant

Comme pour la VR, il s'agit uniquement de la rotation visuelle mais dans le cas où vous jouez sur un ou plusieurs écrans.

Pilote

Cette option permet d'afficher la tête et le corps des pilotes et copilotes. Malheureusement cela ne fonctionne pas avec tous les véhicules pour l'instant.
