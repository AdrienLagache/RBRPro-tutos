Pilote

Dans cet partie vous pouvez retrouver toutes les informations concernant votre profil (nom, pseudo, photo, catégories autorisées, etc.).

C'est ici que vous pouvez envoyer une demande pour changer de Pseudo. Cliquez simplement sur ce dernier.

Le nombre d'étoile est important. C'est ce qui vous permet de débloquer les dernières catégories de véhicules pour les championnats. Pour savoir comment les gagner, consultez le réglement. Le nombre d'étoiles gagnées par position et le nombre d'étoiles nécessaires par catégorie y est donné.
Il est important de préciser également que les étoiles ne sont pas comptées de la même manière si vous jouez seul ou en team.
Vous pouvez évidement utiliser tous les véhicules le reste du temps.

L'onglet "En ligne" permet d'accéder à votre profil CZ mais est voué à disparaître avec l'arrivée du nouveau plugin en ligne RBRPro.

Equipe

La même chose que profil mais concernant l'équipe cette fois-ci.

A noter que si vous faites partie d'une team, ce sera le nombre d'étoiles indiqué sur cette page qui vous autorisera ou non certaines catégories. Il s'agit de la somme du nombre d'étoiles de chaque membre de l'équipe.

Carrière

Il s'agit d'un récapitulatif de vos points sur les différents rallyes de la saison en cours ou même de l'année passée.

Compte

C'est ici que vous devez entrer vos identifiants CZ et Pro, ainsi que l'ID pilote.

Si vous n'étes pas inscrit, rendez-vous dans la rubrique "inscription/donation" de ce site pour savoir comment faire.
