Comment installer RBRPro ?

Pour installer le jeu, rien de plus simple. Il suffit de télécharger la dernière mise à jour sur le site officiel de RBRPpro et d'installer RBRPro Manager.

  1. Téléchargez RBRPro Update. Votre navigateur peut vous afficher un message car l'éditeur est inconnu. Dans ce cas, acceptez le téléchargement.
  2. Exécutez ensuite le programme d'installation
  3. Lorsque le programme vous demande si vous voulez réparer visual C++, sélectionnez oui. Idem pour l'installation de directX 9c.
  4. Et voila ! votre manager est prêt et installé. Lors du premier lancement le programme vous demandera si vous voulez ajouter une exception. Sélectionnez "oui".

Attention : certains antivirus suppriment des fichiers dans le dossier du jeu. Il est recommandé d'ajouter une exception au parefeu Windows ET à votre antivirus. Si cela arrive malgrès tout, vous pourrez toujours les récupérer dans "contenu" puis "pack d'option" dans le Manager.
Enfin, il est recommandé de lancer RBRPro en mode administrateur pour pouvoir profiter de toutes les fonctionnalités.

C'est bon pour vous ? Rendez vous au premier lancement.

Pour lancer le programme en administrateur, il suffit de faire clic-droit dessus puis "exécuter en tant qu'administrateur". Vous pouvez aussi sélectionner propriétés, puis dans l'onglet "Compatibilité", choisissez "Modifier les paramètres pour tous les utilisateurs", et enfin "exécuter ce programme en tant qu'administrateur" (voir ci-dessous).

Premier lancement

Dans le manager :

Après le premier lancement vous devriez arriver dans la rubrique options/manager :

  1. Sélectionnez votre langue
  2. Cliquez sur le bouton "installer un profil de jeu pour cette langue"

Lorsque vous serez inscrit, vous pourrez utiliser la version gratuite de RBRPro. Mais je vous recommande fortement de faire un don afin de profiter de toutes les fonctionnalités et supporter l'équipe de développeurs. Rendez-vous dans la rubrique Inscription/Réglement pour savoir comment faire !
Consultez le manuel de l'utilisateur officiel pour en savoir plus !

Dans le jeu :

Lors du premier lancement il peut y avoir certaines options à modifier, selon vos préférences.

Pour accéder aux options vous devrez revenir en arrière par rapport au menu sur lequel le jeu s'ouvrira. Voici une capture d'écran du menu options :

Les options présentent parlent d'elles même, je vous laisse les découvrir.

Il faut cependant savoir que les options de notes concernent les notes qui s'affichent à l'écran. Vous devrez donc mettre "NON" si vous ne voulez pas les voir. Enfin, la distance d'appel ne concernera que les copilotes standards et non les copilotes navigator. Je vous explique la différence dans le chapitre Contenu/Copilotes.
