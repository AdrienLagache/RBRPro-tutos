C'est ici que vous pourrez régler les différents paramètres de vos addons.

Navigator

Il y a énormement à dire sur cet add-on. C'est pourquoi j'ai créé un chapitre Navigator pour lui seul. Je vous invite à le visiter lorsqu'il sera terminé.

En attendant, je vous invite à suivre ce tuto vidéo. Vous pouvez activer les sous titres en français.

Telemetry Recorder/Telemetry Router

Dans Telemetry Recorder, vous pouvez changer la résolution. Si vous vous rapprochez de 0 vous aurez un maximum de détail mais les performances peuvent en pâtir.

Dans Telemetry Router, vous avez toutes les informations nécessaires pour exploiter la télémétrie avec des add-ons différents de RBRPro. Vous pouvez par exemple utiliser ces données avec fanaled pour afficher les informations de régime moteur, vitesse, essence,... sur votre volant.

WRCDashboard

Le deuxième onglet concerne les mêmes réglages mais pour la VR.

Dans généralité vous allez pouvoir notamment choisir une tolérance pour votre delta. Par exemple laisser l'affichage en vert en dessous d'une seconde de différence. Vous pourrez également modifier l'affichage du régime moteur idéal pour changer de vitesse.

Une fois de plus, je vous invite à consulter le manuel de l'utilisateur WRC PRO Dashboard.
